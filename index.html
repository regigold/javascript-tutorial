step 01

Add Google Maps to Your Webpage

 step 01

Add Google Maps to Your Webpage

First, create a div element named "map" to hold the Map. Assign a width and height to suite your needs.

#map {
    width: 1000px;
    height: 600px;
}

Next, define a JavaScript function that creates a map in the div. You'll also need to include a second script to call the Google Maps API. At this point you'll need to sign up for a unique API number supplied by google. Replace 'YOUR_API_KEY' with the number you receive from google.

<script>
      var map;
      function initMap() {
        map = new google.maps.Map(document.getElementById('map'), {
          center: {lat: 49.2827, lng: -123.1207},
          zoom: 14,
        });
      }
</script>
<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap"
async defer></script>

step 02

Styling Your Google Map

 step 02

Styling Your Google Map

First, let's examine the styling syntax. Below is a break down of the JavaScript phrasing.

var stylesArray = [
  {
    featureType: '',
    elementType: '',
    stylers: [
      {color: ''},
      {visibility: ''},
      // Add any stylers you need.
    ]
  },
  {
    featureType: '',
    // Add the stylers you need.
  }
]

Thanks to the Google API, styling is pretty straight forward. Let's put this syntax to work and try changing the colour of the water in our newly embed map. As you can see below, the 'featureType' targets the water area of the map. Within the water area, we can target a specific 'elementType'. In this case we're targetting 'geometry', 'labels.text.fill' and 'labels.text.stroke' and applying a color styler to each. As you can see in the map on the left, the water, and water labels(fill and stroke) have been updated.

styles: [
            {
              featureType: 'water',
              elementType: 'geometry',
              stylers: [{color: '#17263c'}]
            },
            {
              featureType: 'water',
              elementType: 'labels.text.fill',
              stylers: [{color: '#515c6d'}]
            },
            {
              featureType: 'water',
              elementType: 'labels.text.stroke',
              stylers: [{color: '#17263c'}]
            }
        ]
